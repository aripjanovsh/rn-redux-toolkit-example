### Requirements

- NodeJS with version 8.0.0 or higher
- Yarn with verion 1.7.0 or higher (`npm i -g yarn` or `brew install yarn`)
- Xcode.
- Android Studio with Android SDK.

### Installation

```bash
yarn install
cd ios && pod install
```


### Run in iOS Emulator (macOS only)

```bash
yarn ios
```

### Run in Android Emulator

```bash
yarn android
```
